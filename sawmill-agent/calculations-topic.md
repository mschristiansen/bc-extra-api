# Calculations Topic

## Topic Name
Beregninger og Omregninger (Calculations)

## Topic Description
Hjælper brugeren med at beregne træmængder, omregne mellem enheder (m³, løbende meter, styk), beregne vægt af træ, optimere valg af pakkelængder for at minimere spild, og beregne udbytte ved flækning af brædder. Emnet håndterer også beregning af krumningsreduktion ved afkortning.

This topic helps users with:
- Converting between volume (m³), running meters (lbm), and pieces (stk)
- Calculating weight of wood based on species and volume
- Optimizing package length selection to minimize waste
- Calculating yield when ripping boards into strips
- Estimating warp reduction when cutting boards shorter

## When to Use This Topic
- User asks about converting between m³, meters, or pieces
- User wants to know weight of a specific wood quantity
- User asks about waste/spild calculations
- User wants to optimize cutting for a specific length requirement
- User asks about yield or udbytte from ripping/flækning
- User mentions savsnit (kerf) or sawblade thickness
- User asks about krumning (warp) and cutting

## When NOT to Use This Topic
- User is asking about wood species properties (use wood species topic)
- User is asking about grading or quality
- User is asking about pricing or inventory availability
- User is asking about certification (FSC, PEFC)

## Sample Trigger Phrases
- "Hvor meget vejer 2 m³ eg?"
- "Omregn 100 løbende meter til kubikmeter"
- "Hvor mange emner kan jeg skære fra en 5m pakke?"
- "Beregn spild for 1,8m emner"
- "Hvad er udbyttet hvis jeg flækker 147mm brædder til 65mm lister?"
- "Hvor mange styk i 0,5 m³?"
- "Beregn vægt af 50 lbm 25×100mm Sapelli"
